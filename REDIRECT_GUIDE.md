# Lago Bravo Resort - Redirect Addresses

## GitHub Repository URL
Your main repository: `https://github.com/Bondliden/lagobravo-website`

## GitHub Pages Deployment
After uploading to GitHub, enable GitHub Pages:
1. Go to Settings > Pages
2. Select "Deploy from a branch" 
3. Choose "main" branch
4. Your live site will be: `https://bondliden.github.io/lagobravo-website`

## Custom Domain Options
You can set up custom redirects to:
- `lagobravo.com` (if you own the domain)
- `lago-bravo-resort.com`
- Any custom domain you prefer
