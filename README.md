# Lago Bravo Integrated Resorts

Luxury resort website for Lago Bravo Integrated Resorts featuring championship golf, exclusive marina, crystal lagoon villas, accommodations, weddings, casino, dining, and nightlife.

## Live Site

**Website**: [www.lagobravo.com](https://www.lagobravo.com)  
**Contact**: info@lagobravo.com

## Features

- Modern React 18 frontend with TypeScript
- Express.js backend with PostgreSQL database
- 11-language support with flag-based selector
- Responsive luxury design
- Form validation with Zod
- TanStack Query for state management
